# Use Spring Boot 3

* Status: Accepted

## Problem and Motivation

While "Play Is OK", the amount of play being tried at once needs to be considered.

## Decision

Use Spring Boot 3.

## Consequences

Plentiful integrations, documentation and AI support.

Will need native compilation in order to conserve RAM.

## Rationale

Spring Boot is the most common system for building Java web apps, and I've used it before.
